# React 3D Shape Generator

A modern, interactive 3D shape generator built with React, Three.js, and Tailwind CSS. This application allows users to visualize various geometric shapes, apply textures, and manipulate their dimensions and rotation in real-time.

## Features

- **Shape Selection**: Choose from a wide variety of 3D shapes including Sphere, Cylinder, Cube, Cone, Torus, Pyramids, Prisms, and Platonic solids (Tetrahedron, Dodecahedron, Icosahedron).
- **Real-time Customization**:
  - Adjust Radius and Height.
  - Toggle and adjust segment count for smoother or lower-poly looks.
- **Appearance Controls**:
  - Change base colors.
  - Apply procedural textures: Noise, Checkerboard, Dots, Stripes.
- **Camera & Motion**:
  - Control camera distance (zoom).
  - Adjust rotation speed.
  - Customize rotation axis (X, Y, Z).
- **Responsive UI**: A floating, collapsible settings panel built with Tailwind CSS and glassmorphism effects.

## Tech Stack

- **React 18**: UI and state management.
- **Three.js**: 3D rendering engine.
- **Tailwind CSS**: Styling.
- **Lucide React**: Icons.
- **Vite**: Build tool.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to the local server address.
